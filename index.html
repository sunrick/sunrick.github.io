portfolio

unoverb

this is my 3 week final project that at the iron yard. i worked with a front-end developer. unoverb is a platform for people to practice foreign verb conjugations in a more meaningful way. teachers can create classrooms, create exercises, and see vital statistics on their students' progress. unoverb provides access to 630+ spanish verbs, 18 tenses, with a total of 57,000+ total conjugations. any compatible language can be added to the app and would work immediately.

role: back-end developer

tech used: ruby on rails, cancan, heroku, paperclip, aws3-sdk

links: demo github
route optimization app
