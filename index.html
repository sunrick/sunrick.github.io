smaka.io

smaka.io allows users to listen to popular music around the world without having to listen to the same X amount of popular artists. (i.e. justin bieber) i built the app after being frustrated with not being able to find music that is unique to every country. smaka.io takes the itunes top 100 songs for a bunch of countries and filters out any duplicates it finds across markets. that means if you look at the french music scene, the music is more likely to actually be french!

role: full-stack developer

tech used: ruby on rails, heroku, youtube api, itunes api, bootstrap

links: temporary site
popthatfeeling

personal project i created for people struggling with vocalizing their feelings. basic premise of the app is to type however you're feeling in that moment and pop your feelings away. i built the back-end using ruby on rails and used materialize, paperjs and plain jquery for the front-end of the site.

role: full-stack developer

tech used: ruby on rails, heroku, paperjs, sidekiq + cron, materialize

links: github
unoverb

this is my 3 week final project that at the iron yard. i worked with a front-end developer. unoverb is a platform for people to practice foreign verb conjugations in a more meaningful way. teachers can create classrooms, create exercises, and see vital statistics on their students' progress. unoverb provides access to 630+ spanish verbs, 18 tenses, with a total of 57,000+ total conjugations. any compatible language can be added to the app and would work immediately.

role: back-end developer

tech used: ruby on rails, cancan, heroku, paperclip, aws-sdk, postgresql, jbuilder

links: github
