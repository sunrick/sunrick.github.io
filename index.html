portfolio

unoverb

this is my 3 week final project at the iron yard. unoverb is a platform for people to practice foreign verb conjugations in a more meaningful way. teachers can create classrooms, create exercises, and see vital statistics on their students' progress. unoverb provides access to 630+ spanish verbs, 18 tenses, with a total of 57,000+ total conjugations. any compatible language can be added to the app and would work immediately.

role: back-end developer

tech used: ruby on rails, cancan, heroku, paperclip, aws3-sdk

links: demo github
route optimization app

the iron yard and soltech partnered to host a weekend long hackathon. i worked in a team of 3 front-end developers and one other back-end developer. we were tasked with creating an app where an X amount of addresses could be given by a user in a random order and our app would find the quickest trip possible. i built the route optimization portion of the app. i managed to support a total of 10 addresses using the google maps api with the possiblity of adding 13 more. the app is capable of calculating the fastest route with step by step directions, total duration and also supports different methods of travel.

role: back-end developer

tech used: ruby on rails, heroku, paperclip, aws3-sdk

links: demo github
